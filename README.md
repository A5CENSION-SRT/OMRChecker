# OMRChecker RVCE

### running instructions: 

backend: 

```sh
cd backend
python main.py
```

frontend (localhost:3000): 

```sh
cd omrfront
npm run dev
```

### python virtual env: 

create a venv using uv at the project root and use it for the backend and stresstest as well. 

```sh
uv venv
uv sync
```
